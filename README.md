# 45 Days of DSA 🚀

A personal commitment to solve **3 DSA problems every day for 45 days**, building strong problem-solving fundamentals in **Java** — one commit at a time.

This repo is also my first hands-on journey with **Git & GitHub**, so the workflow itself is part of the learning.

---

## 🎯 Goals

- Solve **3 problems/day, for 45 days** (135+ problems total)
- Build a strong foundation in core DSA before moving to advanced topics
- Get comfortable with Git & GitHub (commits, branches, history) through daily practice
- Create a visible, consistent GitHub contribution streak
- Share the completed journey on LinkedIn 🎉
